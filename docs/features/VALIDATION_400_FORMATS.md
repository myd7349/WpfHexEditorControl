|----------|---------|----------|----------|---------|
| Formats | 72 | 159 | 400 | +152% |
| Catégories | 17 | 20 | 20 | = |
| Extensions | ~150 | ~280 | ~800 | +186% |

| Mémoire JSON | ~1.5MB | ~2.8MB | ~6.5MB | +132% |

**Observation** :
- Le temps de chargement augmente **linéairement** avec le nombre de formats
- Reste < 150ms, ce qui est **excellent** pour 400 formats
- La détection reste **constante** grâce au cache et à la priorité par extension

```csharp
// Ouvrir une IRM
```
