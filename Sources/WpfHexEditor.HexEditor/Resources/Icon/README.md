# Resources/Icon - Icônes du Projet

Icônes et images utilisées dans le projet WPFHexaEditor.

## 📁 Contenu

Fichiers d'icônes (.ico, .png) pour l'application et les contrôles.

## 🖼️ Icônes Disponibles

- **app.ico** - Icône principale de l'application
- Autres icônes pour boutons et contrôles

## 💡 Utilisation

**Dans XAML:**
```xml
<Image Source="/WPFHexaEditor;component/Resources/Icon/app.ico" />
```

**Dans code:**
```csharp
var icon = new BitmapImage(new Uri("pack://application:,,,/WPFHexaEditor;component/Resources/Icon/app.ico"));
```
