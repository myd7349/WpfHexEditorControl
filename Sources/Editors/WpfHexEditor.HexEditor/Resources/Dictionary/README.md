# Resources/Dictionary - Dictionnaires XAML

Dictionnaires de ressources XAML pour styles, templates et brushes.

## 📁 Contenu

Dictionnaires de styles réutilisables pour l'interface du HexEditor.

## 🎨 Types de Ressources

- **Styles** - Styles pour contrôles WPF
- **Templates** - Control templates personnalisés
- **Brushes** - Couleurs et gradients
- **Converters** - Convertisseurs de valeurs

## 💡 Utilisation

```xml
<ResourceDictionary Source="/WPFHexaEditor;component/Resources/Dictionary/Styles.xaml" />
```
